# 🌐 Vitalisyshak Tech LLP Website

This is the official website for **Vitalisyshak Tech LLP**, a tech company showcasing our services, projects, and company information. The website is responsive, interactive, and built to perform well across all devices.

## 🔗 Live Demo

[https://kana-ram.github.io/vitalisyshaktech](https://kana-ram.github.io/vitalisyshaktech) (if deployed on GitHub Pages)

## 📌 Features

- ✅ Modern responsive design
- ✅ Clean single-page layout
- ✅ Scroll animations and interactivity
- ✅ Contact form integrated with Firestore
- ✅ SEO-optimized for "vital", "vitali", and "vitalisyshak"
- ✅ Firebase-ready architecture
- ✅ Developed using React (converted from static HTML/CSS/JS)
